# Music Player

A simple Music Player built with **HTML**, **CSS**, and **JavaScript**.

## Features

* Play and pause songs
* Previous and next track navigation
* Dynamic song title and album cover
* Rotating album artwork while music is playing
* Interactive progress bar
* Automatically plays the next song when the current one

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
